## 리소스 정리 과정

:::caution[시작하기 전에]
리소스 제거가 확실하지 않은 경우 유지하는 것을 고려하세요. 향후 사용을 위해 가치가 있을 수 있습니다.
:::

<Steps>
1. **CDK 스택 삭제**
   ```bash
   pnpm cdk_infra:destroy
   ```
   :::note
   이 명령은 확인 프롬프트 없이 CDK 애플리케이션을 삭제합니다.
   :::

2. **리소스 삭제 확인**
   - [CloudFormation 콘솔](https://console.aws.amazon.com/cloudformation)로 이동
   - 모든 스택이 제거되었는지 확인
   - 삭제 실패가 있는지 확인

3. **수동 정리(필요한 경우)**
   - 남아있는 리소스 강제 삭제
   - 관련 서비스에서 남아있는 구성 요소 확인
</Steps>
