이 워크샵은 Amazon Bedrock Knowledge Bases와 Agents를 사용하여 AI 기반 애플리케이션을 빠르게 프로토타이핑하기 위한 CDK 스캐폴딩을 제공합니다.

이 풀스택 템플릿에는 사전 구축된 인프라, 샘플 구현 및 사용자 정의 가능한 컴포넌트가 포함되어 있어 AWS를 기반으로 한 자체 생성형 AI 솔루션을 신속하게 구축하고 검증할 수 있습니다.

## 목적

* 프로덕션 준비가 된 인프라 템플릿으로 개발 가속화
* 맞춤형 사용 사례를 위한 빠른 개념 증명 개발 지원
* Amazon Bedrock 서비스 통합을 위한 모범 사례 시연

## 사전 요구사항

### 필수 AWS 설정

<Steps>
  1. **AWS 계정 액세스**
     - AWS 계정에 대한 관리자 액세스 권한 확인

  2. **Amazon Bedrock 모델 액세스**
     - Bedrock Agent용 Claude (Sonnet 3, 3.5)
     - Knowledge Base용 Titan Embeddings v2
     - 채팅 요약용 Nova Lite (선택사항)

  3. **CDK 부트스트랩**
     - 대상 리전에서 CDK 부트스트랩 실행 (권장: `us-west-2` 또는 `us-east-1`)
     ```bash
     cdk bootstrap aws://ACCOUNT-NUMBER/REGION
     ```
     :::note
     다른 리전을 사용하는 경우 [AWS 서비스](/applications/cdk-infra-guide)와 [기반 모델](/getting-started/#required-aws-setup)이 선택한 리전에서 지원되는지 확인하세요.
     :::
</Steps>

## 저장소 구조

이 프로젝트는 [PNPM Workspaces](https://pnpm.io/workspaces)로 관리되는 모노레포로, 단일 저장소에 여러 관련 프로젝트가 포함되어 있습니다.

<FileTree>
- bedrock-agent-knowledge-base-actions-workshop/
  - docs/ 문서화 및 참조 자료
  - tools/ 개발 도구
  - packages/
    - cdk_infra/ 백엔드 인프라 프로젝트 (AWS CDK)
    - reactjs_ui/ 프론트엔드 샘플 애플리케이션 (ReactJs + Vite)
  - pnpm-lock.yaml PNPM 의존성 잠금 파일
  - pnpm-workspace.yaml PNPM 워크스페이스 정의 파일
  - package.json 프로젝트 메타데이터 및 스크립트
  - requirements.txt Python 개발 의존성
  - LICENSE 소프트웨어 라이선스
</FileTree>
